1. 克隆 `QEMU` 代码，运行

```bash varcode
git clone ${_http}://${_domain}/git/qemu.git
```

2. 将 `mirror` 加入已有代码库，在已有仓库中运行

```bash varcode
git remote add mirror ${_http}://${_domain}/git/qemu.git
```

或

```bash varcode
git remote set-url origin ${_http}://${_domain}/git/qemu.git
```

将默认上游设置为当前软件镜像站。
